组成

通用样式库由几个目录组成


+ config
    + config.less  网站配置存放, 每个项目都需要做适量的修改
    + func.less    常用函数配置

+ core
    + rockstar.less 这个就用公司名，命名把...

    这个就是核心库，最后合并lib目录的所有文件

    生成rockstar.css, 然后页面引用即可

+ extend
    + css3.less        css3扩展,以后任何扩展都可以放这里面

+ font-icon
    + icon.less
    + icon-ie7.less     字体图标库

+ lib
    + reset.less     重置
    + base.less      基础样式
    + bg.less        背景
    + button.less    用样式写出来的按钮
    + form.less      表单
    + table.less     表格
    + help.less      辅助样式
    + img-button.less    使用图标的按钮
    + img-icon.less      使用icon的按钮
    + img.less           图片样式
    + layer.less         网站布局
    + size.less          宽高尺寸定义

+ main
    +... 一个页面对应一个less文件


+ widget
    +... 页面组件

表单

独占一行的form-control元素。input如果不设置宽度默认是98%,ie7不支持box-sizing:border-box,内边距和边框要占宽度。ie7 ie8无圆角,focus无渐变边框，shadow。 form-control默认没有高度，上下默认7px的内边距。所有配置均可在config.less里面配置。


.form-control {    
  display: block;
  width: 100% - @form-base-left-padding;
  padding: @form-base-vertical-padding 0;
  padding-left: @form-base-left-padding;
  font-size: @form-base-size;
  color: @form-base-color;
  border: 1px solid @form-base-border-color;
  border-radius: @form-base-radius;
  background-color: @form-base-bg;

  .transition(all .15s ease-in-out);
  .form-control-focus(@form-focus-border);
  .placeholder(@form-placeholder-color);

  &[disabled],
  &[readonly],
  &.disabled,
  &.readonly {
    background-color: @form-disabled-color;
    opacity: 1;
    .form-control-focus(@form-base-border-color);
    box-shadow: none@i;
  }

}
